# Kalkulator

Jednostavna web aplikacija za osnovne aritmeticke operacije: sabiranje, oduzimanje, mnozenje i dijeljenje.

## Live verzija
[https://vite-react-calculator.vercel.app](https://vite-react-calculator.vercel.app)

## Tehnologije
- React
- Vite
- TypeScript

## Pokretanje lokalno

```bash
npm install
npm run dev
```
